# NLP Text Processing and Classification Projects

This repository contains multiple Jupyter notebooks demonstrating important Natural Language Processing (NLP) concepts including text preprocessing, text representation, word embeddings, and sentiment classification using the IMDB dataset.

---

## Repository Structure

### Files Included

| File Name                        | Description                                                          |
| -------------------------------- | -------------------------------------------------------------------- |
| `Data.zip`                       | Dataset files used in the notebooks                                  |
| `Text_Preprocessing_Step.ipynb`  | Covers text preprocessing techniques used in NLP                     |
| `Text_representation.ipynb`      | Demonstrates different text vectorization and representation methods |
| `word2Vec_GameofThrones.ipynb`   | Word2Vec implementation using Game of Thrones text data              |
| `Text_classification_IMDB.ipynb` | IMDB movie review sentiment classification project                   |

---

# Project Overview

## 1. Text Preprocessing

Notebook: `Text_Preprocessing_Step.ipynb`

This notebook demonstrates essential NLP preprocessing techniques such as:

* Lowercasing
* Tokenization
* Stopword Removal
* Stemming
* Lemmatization
* Removing Punctuation
* Removing Special Characters
* Text Cleaning

These steps are important before building NLP models.

---

## 2. Text Representation

Notebook: `Text_representation.ipynb`

This notebook explains how textual data can be converted into numerical form using:

* Bag of Words (BoW)
* N-Grams
* TF-IDF (Term Frequency - Inverse Document Frequency)
* Count Vectorization

These representations are commonly used in machine learning models for NLP tasks.

---

## 3. Word2Vec using Game of Thrones Dataset

Notebook: `word2Vec_GameofThrones.ipynb`

This project demonstrates:

* Training Word2Vec embeddings
* Semantic similarity between words
* Word vector visualization
* Contextual understanding of words

Dataset/theme used:

* Game of Thrones text corpus

Libraries used:

* Gensim
* NumPy
* Pandas
* Matplotlib

---

## 4. IMDB Sentiment Classification

Notebook: `Text_classification_IMDB.ipynb`

This notebook implements sentiment analysis on the IMDB movie review dataset.

### Features:

* Data preprocessing
* Feature extraction
* Text vectorization
* Model training
* Accuracy evaluation

### Algorithms Used:

* Logistic Regression
* Naive Bayes
* Machine Learning based NLP models

### Objective:

Classify movie reviews as:

* Positive
* Negative
